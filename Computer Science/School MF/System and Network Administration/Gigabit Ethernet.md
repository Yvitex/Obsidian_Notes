# Gigabit Ethernet
A type of [[Ethernet]] that runs about 10000 mbps speed and the most expensive type of ethernet. This is commonly used to connect [[Server Computer]] to the [[Network]]

Gigabit ethernet types:
- [[1000BaseT]]
- [[1000BaseLX]]
