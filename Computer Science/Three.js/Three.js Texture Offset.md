# Texture Offset
Offset will move the texture relative to its current position. 
![[Pasted image 20220816060248.png]]

```js
colorTexture.offset.x = 0.5;
```
![[Pasted image 20220816084035.png]]
```js
colorTexture.offset.x = 0.5;
colorTexture.offset.y = 0.5;
```
