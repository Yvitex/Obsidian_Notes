# FlexBox
Is a useful too for website responsivleness in [[CSS]] 

The basic usage is 
```css
img-container {
	display: flex;
	flex-wrap: wrap;
}
```

This way, the children is shifted to a new line. when the viewport shrink. We could also use it to center items
```css
img-container {
	display: flex;
	flex-wrap: wrap;
	justify-content: center;
}
```
