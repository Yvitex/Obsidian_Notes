## Greensock Object
The core of green sock is the greensock object, which is the access point for every animation and etc.
Greensock object have 3 main [[OOP|methods]] namely
- `gsap.to()`
- `gsap.from()`
- `gsap.fromTo()`

This constrols tha animation creating a timeline.
